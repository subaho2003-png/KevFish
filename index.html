Data Sources

Open-Meteo Forecast API

Precipitation probability (%) and max wind speed (mph) for up to 16 days. Free, no API key required. Used for rain & wind scores.

Open-Meteo Marine API

Offshore significant wave height (converted from meters to feet) for ~7–8 days. Free, no key required. Primary wave source.

Windy Point Forecast API

GFS model wave height extended out to 16 days. Used as fallback when Open-Meteo Marine has no wave data (days 9+). Free API key required — register at windy.com.

NWS Marine Alerts (api.weather.gov)

Active NOAA marine alerts for zone AMZ630 — Coastal Waters from Sebastian Inlet to Jupiter Inlet FL, 0–20 NM. Displayed as a red banner when active.

Open-Meteo Historical API

ERA5 reanalysis for past precipitation totals and wind speeds. Used on the Actuals screen. Data has a ~2 day lag.

NOAA NDBC Buoy 41114

Real-time buoy near Fort Pierce (27.551°N, 80.225°W). Provides ground truth wave height and conditions. Planned for future wave actuals integration.
